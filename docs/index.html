Pi-Hole en Docker

Aprender a usar docker El objetivo principal de este proyecto es mostrar el uso de docker. Es una aplicación extremadamente sencilla de poner en marcha y muy práctica en un entorno personal.

Disponer de un sistema de eliminación de publicidad. La puesta en marcha del servicio nos permitirá eliminar en nuestra red local (LAN) la publicidad que aparece en la navegación en nuestros dispositivos.

Disponer de un servicio DNS propio para nuestra red. Gracias a la funcionalidad adicional, podemos asignar nombres a direcciones IP estáticas de nuestra red local. Esto facilita la identificación y acceso a los dispositivos de nuestra red.
